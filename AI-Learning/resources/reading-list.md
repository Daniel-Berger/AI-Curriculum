# Curated Reading List

A comprehensive reading list organized by learning phase. Resources marked with (free) are available at no cost.

---

## Phase 1: Foundations

### Python

| Resource | Type | Notes |
|----------|------|-------|
| [Official Python Tutorial](https://docs.python.org/3/tutorial/) | Docs (free) | Start here if new to Python |
| *Fluent Python* - Luciano Ramalho | Book | Advanced Python idioms, 2nd edition covers 3.10+ |
| *Python Cookbook* - David Beazley & Brian K. Jones | Book | Recipe-based approach for intermediate devs |
| [Real Python](https://realpython.com/) | Website (free/paid) | Excellent tutorials on every Python topic |

### Mathematics

| Resource | Type | Notes |
|----------|------|-------|
| [3Blue1Brown - Essence of Linear Algebra](https://www.youtube.com/playlist?list=PLZHQObOWTQDPD3MizzM2xVFitgF8hE_ab) | Video (free) | Visual intuition for vectors, matrices, eigenvalues |
| [3Blue1Brown - Essence of Calculus](https://www.youtube.com/playlist?list=PLZHQObOWTQDMsr9K-rj53DwVRMYO3t5Yr) | Video (free) | Derivatives, integrals, chain rule visually explained |
| [Khan Academy - Statistics & Probability](https://www.khanacademy.org/math/statistics-probability) | Course (free) | Distributions, hypothesis testing, Bayes theorem |
| *Mathematics for Machine Learning* - Deisenroth, Faisal, Ong | Book (free) | [Free PDF available](https://mml-book.github.io/) |

### NumPy & Pandas

| Resource | Type | Notes |
|----------|------|-------|
| [NumPy Official Docs](https://numpy.org/doc/stable/) | Docs (free) | Focus on broadcasting, vectorization, indexing |
| [Pandas Official Docs](https://pandas.pydata.org/docs/) | Docs (free) | User guide + cookbook sections |
| *Python for Data Analysis* - Wes McKinney | Book | Written by pandas creator, covers full data workflow |
| [Kaggle Learn - Pandas](https://www.kaggle.com/learn/pandas) | Course (free) | Hands-on micro-course with exercises |

### Machine Learning

| Resource | Type | Notes |
|----------|------|-------|
| [Andrew Ng - ML Specialization](https://www.coursera.org/specializations/machine-learning-introduction) | Course (free audit) | Gold standard intro: regression, classification, neural nets |
| *Hands-On Machine Learning* - Aurelien Geron | Book | Scikit-learn + TensorFlow/Keras, very practical |
| [Scikit-learn Documentation](https://scikit-learn.org/stable/user_guide.html) | Docs (free) | Excellent user guide with theory + code |
| [Kaggle Learn Courses](https://www.kaggle.com/learn) | Course (free) | Intro ML, Intermediate ML, Feature Engineering |
| [StatQuest with Josh Starmer](https://www.youtube.com/c/joshstarmer) | Video (free) | Clear, visual explanations of ML algorithms |

### Data Visualization

| Resource | Type | Notes |
|----------|------|-------|
| [Matplotlib Gallery](https://matplotlib.org/stable/gallery/index.html) | Docs (free) | Copy-paste examples for every chart type |
| [Seaborn Gallery](https://seaborn.pydata.org/examples/index.html) | Docs (free) | Statistical visualization examples |
| *Storytelling with Data* - Cole Nussbaumer Knaflic | Book | How to communicate effectively with charts |
| [Python Graph Gallery](https://python-graph-gallery.com/) | Website (free) | Searchable gallery with code snippets |

## Phase 2: Deep Learning & NLP

### Deep Learning

| Resource | Type | Notes |
|----------|------|-------|
| [Fast.ai - Practical Deep Learning](https://course.fast.ai/) | Course (free) | Top-down approach, get results fast, then understand why |
| [PyTorch Tutorials](https://pytorch.org/tutorials/) | Docs (free) | Official tutorials from tensors to deployment |
| *Deep Learning* - Ian Goodfellow, Yoshua Bengio, Aaron Courville | Book (free) | [deeplearningbook.org](https://www.deeplearningbook.org/) - the theory bible |
| [Neural Networks and Deep Learning](http://neuralnetworksanddeeplearning.com/) - Michael Nielsen | Book (free) | Excellent intuitive online book with interactive elements |
| [Dive into Deep Learning](https://d2l.ai/) | Book (free) | Interactive book with code in PyTorch, TensorFlow, JAX |

### NLP

| Resource | Type | Notes |
|----------|------|-------|
| [HuggingFace NLP Course](https://huggingface.co/learn/nlp-course) | Course (free) | Transformers, tokenizers, fine-tuning with HF ecosystem |
| [The Illustrated Transformer](https://jalammar.github.io/illustrated-transformer/) - Jay Alammar | Blog (free) | Best visual explanation of transformer architecture |
| *Speech and Language Processing* - Jurafsky & Martin | Book (free) | [web.stanford.edu/~jurafsky/slp3/](https://web.stanford.edu/~jurafsky/slp3/) - comprehensive NLP textbook |
| [Attention Is All You Need](https://arxiv.org/abs/1706.03762) | Paper (free) | The original transformer paper - essential reading |
| [The Illustrated BERT](https://jalammar.github.io/illustrated-bert/) - Jay Alammar | Blog (free) | Visual walkthrough of BERT, ELMo, GPT |

### Videos

| Resource | Type | Notes |
|----------|------|-------|
| [3Blue1Brown - Neural Networks](https://www.youtube.com/playlist?list=PLZHQObOWTQDNU6R1_67000Dx_ZCJB-3pi) | Video (free) | Visual intuition for neural nets and backpropagation |
| [Andrej Karpathy - Neural Networks: Zero to Hero](https://www.youtube.com/playlist?list=PLAqhIrjkxbuWI23v9cThsA9GvCAUhRvKZ) | Video (free) | Build GPT from scratch, masterful teaching |
| [MIT 6.S191 - Intro to Deep Learning](https://www.youtube.com/playlist?list=PLtBw6njQRU-rwp5__7C0oIVt26ZgjG9NI) | Video (free) | Concise, annually updated MIT course |

## Phase 3: LLMs, RAG & Agents

### LLMs

| Resource | Type | Notes |
|----------|------|-------|
| [Anthropic API Documentation](https://docs.anthropic.com/) | Docs (free) | Claude API reference, guides, and best practices |
| [OpenAI API Documentation](https://platform.openai.com/docs) | Docs (free) | GPT API reference and cookbook |
| *What Is ChatGPT Doing and Why Does It Work?* - Stephen Wolfram | Book/Blog (free) | Accessible explanation of LLM mechanics |
| [Andrej Karpathy - Let's build GPT](https://www.youtube.com/watch?v=kCc8FmEb1nY) | Video (free) | Build GPT-2 from scratch in 2 hours |
| [LLM Visualization](https://bbycroft.net/llm) | Interactive (free) | 3D visualization of transformer internals |

### Prompt Engineering

| Resource | Type | Notes |
|----------|------|-------|
| [Anthropic Prompt Engineering Guide](https://docs.anthropic.com/en/docs/build-with-claude/prompt-engineering) | Docs (free) | Claude-specific prompt techniques |
| [OpenAI Prompt Engineering Guide](https://platform.openai.com/docs/guides/prompt-engineering) | Docs (free) | Best practices for GPT models |
| [DAIR.AI Prompt Engineering Guide](https://www.promptingguide.ai/) | Website (free) | Comprehensive community-maintained guide |
| [Anthropic Cookbook](https://github.com/anthropics/anthropic-cookbook) | GitHub (free) | Practical prompt engineering examples |

### RAG (Retrieval-Augmented Generation)

| Resource | Type | Notes |
|----------|------|-------|
| [LangChain Documentation](https://python.langchain.com/docs/) | Docs (free) | Framework for building RAG pipelines |
| [LlamaIndex Documentation](https://docs.llamaindex.ai/) | Docs (free) | Data framework for LLM applications |
| [RAGAS Documentation](https://docs.ragas.io/) | Docs (free) | RAG evaluation framework |
| [RAG for Knowledge-Intensive NLP Tasks](https://arxiv.org/abs/2005.11401) | Paper (free) | The original RAG paper by Lewis et al. |
| [Pinecone Learning Center](https://www.pinecone.io/learn/) | Website (free) | Vector DB concepts, embedding models, RAG patterns |
| [ChromaDB Docs](https://docs.trychroma.com/) | Docs (free) | Lightweight open-source vector database |

### Agents

| Resource | Type | Notes |
|----------|------|-------|
| [LangGraph Documentation](https://langchain-ai.github.io/langgraph/) | Docs (free) | Framework for building stateful agents |
| [ReAct: Synergizing Reasoning and Acting](https://arxiv.org/abs/2210.03629) | Paper (free) | Foundational agent reasoning pattern |
| [Model Context Protocol (MCP) Specification](https://modelcontextprotocol.io/) | Spec (free) | Anthropic's protocol for tool use |
| [Building Effective Agents](https://www.anthropic.com/research/building-effective-agents) | Blog (free) | Anthropic's guide to agent architectures |
| [AutoGPT](https://github.com/Significant-Gravitas/AutoGPT) | GitHub (free) | Reference implementation of autonomous agents |
